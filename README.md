# Pulsentro

**Pulsentro** is a free, open-source web application that helps users better understand heart-rate recovery after physical activity. By connecting to compatible Bluetooth heart-rate wearables, Pulsentro transforms real-time heart-rate data into clear, interactive visualizations that make cardiovascular recovery easier to explore and understand.

## Why Pulsentro?

Most fitness apps focus on metrics like steps, calories, and workout intensity. Pulsentro takes a different approach by focusing on **heart-rate recovery**—how your heart responds after exercise. The goal is to make this information more accessible through intuitive visualizations and educational insights.

## Features

* Connect to compatible Bluetooth heart-rate monitors using the Web Bluetooth API
* View live heart-rate data in real time
* Record heart-rate recovery sessions after physical activity
* Visualize recovery through interactive charts and graphs
* Compare previous recovery sessions over time
* Store recovery sessions locally in the browser (Version 1)
* Modern, responsive interface built for desktop and mobile

## Technology

* Next.js
* TypeScript
* Tailwind CSS
* Web Bluetooth API
* Local Storage
